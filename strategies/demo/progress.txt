## Technical Notes

### Key Imports
```python
from src.strategies import IntentStrategy, MarketSnapshot, almanak_strategy
from src.intents import Intent
```

### Intent Types Available
- Intent.swap() - Exchange tokens
- Intent.hold() - Do nothing
- Intent.supply() - Supply to lending protocol
- Intent.borrow() - Borrow from lending protocol
- Intent.repay() - Repay borrowed tokens
- Intent.lp_open() - Open LP position
- Intent.lp_close() - Close LP position
- Intent.perp_open() - Open perpetual position
- Intent.perp_close() - Close perpetual position
- Intent.bridge() - Bridge tokens between chains
- Intent.sequence() - Multi-step intent sequence

### Anvil Test Script Pattern (CLI-based)
```python
# 1. Start Anvil fork
anvil = AnvilManager(fork_url, port)
anvil.start()

# 2. Fund wallet with required tokens
fund_wallet_with_usdc(wallet, amount)
fund_wallet_with_weth(wallet, amount)

# 3. Set environment variables for CLI
env = os.environ.copy()
env["ALMANAK_CHAIN"] = "arbitrum"
env["ALMANAK_RPC_URL"] = "http://127.0.0.1:8545"
env["ALMANAK_PRIVATE_KEY"] = ANVIL_PRIVATE_KEY

# 4. Run strategy via CLI
subprocess.run([
    "python", "-m", "src.cli.run",
    "--strategy", "demo_uniswap_lp",
    "--config", config_path,
    "--once",
], env=env)

# 5. Stop Anvil
anvil.stop()
```
