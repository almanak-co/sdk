- Pool: WETH/USDC/3000 (0.3% fee tier)
- Base Range Width: 10%
- Amounts: 0.002 WETH + 3 USDC (~$6 total)
- Supported Chains: Arbitrum, Base, Optimism, Ethereum

**Strategy Logic:**
1. Compute ATR(14) of WETH price
2. Calculate adaptive range: base_width * (1 + ATR/price)
3. Volatility regimes:
   - LOW (ATR < 2%): ~5% width
   - MEDIUM (ATR 2-5%): ~10% width
   - HIGH (ATR > 5%): ~15% width
4. Rebalance on: regime change OR price exits range
5. Tracks _current_volatility_regime in state

**LP Position Details:**
- NFT-based position (Uniswap V3)
- Tick range computed from price and adaptive width
- Position tracked via token_id in state

**Dashboard Features:**
- Current ATR value and percentage
