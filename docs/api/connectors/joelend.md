# Joe Lend (Banker Joe) — DORMANT

> **DEPRECATED — DO NOT USE.** Joe Lend (the lending arm of Trader Joe / LFJ on
> Avalanche) was wound down by its governance in 2026. All on-chain
> supply/borrow/repay/withdraw calls revert with `Error: wind down`. The
> connector has been removed from the Almanak SDK's officially supported
> protocol surface; instantiating `JoeLendAdapter` raises
> `JoeLendDeprecatedError`. The receipt parser is retained for historical
> receipts only. Full removal is tracked for July (VIB-3960).
