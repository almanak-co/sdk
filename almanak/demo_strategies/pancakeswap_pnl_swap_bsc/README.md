# Pancakeswap Pnl Swap Bsc (Demo)

PancakeSwap V3 RSI swap for PnL backtesting on BSC — buy/sell WBNB based on RSI

## Chain

bsc

## Quick Start

```bash
almanak strat demo --name pancakeswap_pnl_swap_bsc
cd pancakeswap_pnl_swap_bsc
almanak strat run --network anvil --once
```

## Configuration

Edit `config.json` to adjust strategy parameters. See `strategy.py` for details.
