# Pancakeswap Swap Bsc (Demo)

PancakeSwap V3 swap lifecycle (BUY/SELL) on BSC

## Chain

bsc

## Quick Start

```bash
almanak strat demo --name pancakeswap_swap_bsc
cd pancakeswap_swap_bsc
almanak strat run --network anvil --once
```
