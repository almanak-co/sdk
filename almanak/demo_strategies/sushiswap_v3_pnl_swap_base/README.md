# Sushiswap V3 Pnl Swap Base (Demo)

SushiSwap V3 RSI swap for PnL backtesting on Base -- buy/sell WETH based on RSI

## Quick Start

```bash
almanak strat demo --name sushiswap_v3_pnl_swap_base
cd sushiswap_v3_pnl_swap_base
almanak strat run --network anvil --once
```
