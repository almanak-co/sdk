# Solana Meme Screener (Demo)

DexScreener-powered meme coin momentum screener on Solana

## Chain

solana

## Quick Start

```bash
almanak strat demo --name solana_meme_screener
cd solana_meme_screener
almanak strat run --once
```

Edit `config.json` to adjust strategy parameters. See `strategy.py` for details.
