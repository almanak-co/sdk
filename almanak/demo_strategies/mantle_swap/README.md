# Mantle Swap (Demo)

Basic swap demo on Mantle L2 — USDT -> WMNT via Agni Finance

## Chain

mantle

## Quick Start

```bash
almanak strat demo --name mantle_swap
cd mantle_swap
almanak strat run --network anvil --once
```
