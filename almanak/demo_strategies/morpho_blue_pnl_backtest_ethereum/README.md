# Morpho Blue Pnl Backtest Ethereum (Demo)

Morpho Blue wstETH collateral supply strategy for PnL backtesting on Ethereum

## Chain

ethereum

## Quick Start

```bash
almanak strat demo --name morpho_blue_pnl_backtest_ethereum
cd morpho_blue_pnl_backtest_ethereum
almanak strat run --network anvil --once
```

## Configuration

Edit `config.json` to adjust strategy parameters. See `strategy.py` for details.
