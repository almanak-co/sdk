# Compound V3 Pnl Backtest Ethereum (Demo)

Compound V3 supply rate tracker on Ethereum for PnL backtesting

## Chain

ethereum

## Quick Start

```bash
almanak strat demo --name compound_v3_pnl_backtest_ethereum
cd compound_v3_pnl_backtest_ethereum
almanak strat run --network anvil --once
```

## Configuration

Edit `config.json` to adjust strategy parameters. See `strategy.py` for details.
