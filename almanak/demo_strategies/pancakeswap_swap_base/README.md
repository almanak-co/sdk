# PancakeSwap Swap Base (Demo)

PancakeSwap V3 swap lifecycle (BUY/SELL) on Base

## Chain

base

## Quick Start

```bash
almanak strat demo --name pancakeswap_swap_base
cd pancakeswap_swap_base
almanak strat run --network anvil --once
```
