# Fluid Swap Arb (Demo)

Fluid DEX stablecoin swap on Arbitrum — trades USDC/USDT on Fluid DEX

## Chain

arbitrum

## Quick Start

```bash
almanak strat demo --name fluid_swap_arb
cd fluid_swap_arb
almanak strat run --network anvil --once
```
