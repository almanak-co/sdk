# Aerodrome Rsi (Demo)

RSI swap strategy on Base via Aerodrome — PnL backtest vehicle

## Chain

base

## Quick Start

```bash
almanak strat demo --name aerodrome_rsi
cd aerodrome_rsi
almanak strat run --network anvil --once
```

## Configuration

Edit `config.json` to adjust strategy parameters. See `strategy.py` for details.
