# Pancakeswap Lp (Demo)

PancakeSwap V3 LP lifecycle on Arbitrum (LP_OPEN + LP_CLOSE)

## Chain

arbitrum

## Quick Start

```bash
almanak strat demo --name pancakeswap_lp
cd pancakeswap_lp
almanak strat run --network anvil --once
```
