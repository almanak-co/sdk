# Meteora Dlmm Lp (Demo)

Open Meteora DLMM LP position on Solana (demo)

## Chain

solana

## Quick Start

```bash
almanak strat demo --name meteora_dlmm_lp
cd meteora_dlmm_lp
almanak strat run --once
```

## Configuration

Edit `config.json` to adjust strategy parameters. See `strategy.py` for details.
