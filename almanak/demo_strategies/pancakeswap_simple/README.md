# Pancakeswap Simple (Demo)

Simple WETH->USDC swap on PancakeSwap V3 for testing

## Chain

arbitrum

## Quick Start

```bash
almanak strat demo --name pancakeswap_simple
cd pancakeswap_simple
almanak strat run --network anvil --once
```

## Configuration

Edit `config.json` to adjust strategy parameters. See `strategy.py` for details.
